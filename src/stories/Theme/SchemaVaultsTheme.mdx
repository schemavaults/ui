# @schemavaults/theme

The [`@schemavaults/theme` package (https://github.com/schemavaults/theme)](https://github.com/schemavaults/theme) generates a TailwindCSS config equipped with TailwindCSS plugins, required colors, etc. needed to make all the `@schemavaults/ui` components function properly and look beautiful.
